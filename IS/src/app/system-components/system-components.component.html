System  Components

THE METERS

The data loggers which can be fitted on both new and existing systems collect user electrical and water data, which they then feed to the web portal for processing

IN-HOUSE DISPLAY

The portal can be installed, locked and viewed through an in-house display to be installed at each unit or house.. The system is accessible through the web via a wifi or mobile

THE WEB PORTAL

The web portal has two user types, the administrator and the home/unit owner. From here the owner is able to view their individual water and electricity usage,
